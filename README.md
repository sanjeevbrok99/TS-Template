# The RESUMEBEAR

## Stacks

- TypeScript

- Node js
- Express js
- MongoDB
- Cloudinary (for image hosting)

## Server Installation

```
cd server
npm install or yarn install
```

#### Add .env file to root server directory

```
DATABASE_URI=
JWT_SECRET=
GOOGLE_WEB_CLIENT_ID=
GOOGLE_ANDROID_CLIENT_ID=
GOOGLE_CLIENT_SECRET=
CLIENT_PUBLIC_URL=
CLOUDINARY_NAME=
CLOUDINARY_API_KEY=
CLOUDINARY_API_SECRET=
```

### This will run server

```
npm run dev
```
